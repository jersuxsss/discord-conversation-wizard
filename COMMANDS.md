# Quick Start Commands

## Development

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Run tests
npm test

# Run tests in watch mode
npm test:watch

# Generate coverage report
npm test:coverage

# Lint code
npm run lint

# Format code
npm run format
```

## Publishing

```bash
# Run pre-publish checks
npm run prepublish:check

# Publish to NPM
npm publish
```

## Examples

```bash
# Run basic example
npx ts-node examples/basic-wizard.ts

# Run advanced example
npx ts-node examples/advanced-wizard.ts
```
